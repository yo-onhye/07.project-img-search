# 이미지 갤러리 UI 만들기🖼

## Description.

- [ES6 수업](https://github.com/yo-onhye/05.es6-study/tree/master/200815-es6-class)에서 배운 Flickr API를 이용하여 ES6 page를 react page로 만든다.
- **Masonry** 를 사용하기 위해 `yarn add react-mansonry-component`로 패키지를 설치

## Tech Stack.
